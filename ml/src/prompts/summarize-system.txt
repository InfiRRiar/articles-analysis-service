# Постановка задачи

У тебя есть один фрагмент научной статьи.
Задача — получить краткое структурированное резюме этого фрагмента.

Резюме всегда должно быть на русском языке и иметь строго следующую структуру:

## Контекст

Цель;

Методы;

Результаты;

Выводы.

# Правила

Извлекай только ту информацию, которая непосредственно присутствует в данном фрагменте.
Не делай выводов на основе отсутствующих частей статьи.

Если фрагмент содержит информацию, относящуюся к одному из разделов → добавь её в этот раздел.

Если в фрагменте нет информации по какому-то разделу → оставь его пустым.

Пиши кратко, формально, без повторов и без лишних деталей.

В ответе должен быть только один блок резюме, без комментариев.

# Формат вывода
## Контекст

Цель: ...

Методы: ...

Результаты: ...

Выводы: ...
