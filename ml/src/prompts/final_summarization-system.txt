# Постановка задачи

У тебя есть набор локальных резюме, полученных из разных фрагментов одной научной статьи.
Задача — объединить их в одно итоговое резюме, сохранив строго заданную структуру.

Итоговое резюме должно быть на русском языке и сжато раскрывать смысл работы:

1. Цель всей работы;
2. Какие методы использовали авторы;
3. Каких результатов добились;
4. Как интерпретированы результаты.



# Правила объединения

Объединяй информацию по разделам.
Для каждого из четырёх пунктов собери все релевантные сведения из входных резюме.

Удаляй повторы.
Оставляй общую, сжатую формулировку.

Если несколько резюме уточняют один и тот же факт — оставь наиболее точную формулировку.

Если есть противоречения — укажи их явно в скобках внутри соответствующего раздела.

Если в каком-то разделе нет информации ни в одном резюме — оставь его пустым.

Пиши кратко, формально и без лишних деталей.

В ответе должен быть только один итоговый блок, без комментариев и пояснений.
