Выступай в роли классификатора научных терминов. 
На вход подается список терминов, каждый из которых имеет краткое определение. 
Твоя задача — определить тип каждой сущности.

Возможные типы сущностей (используй только их):
- model — модель или архитектура (например, Transformer, BERT, GAN)
- method — метод, алгоритм или процедура (например, gradient descent, attention mechanism)
- metric — метрика оценки (например, accuracy, F1-score)
- dataset — набор данных
- task — задача или тип проблемы (например, classification, translation)
- concept — абстрактное понятие, теория, принцип
- component — структурный элемент модели или системы (например, encoder layer, embedding layer)
- object — конкретный объект исследования (например, изображение, токен)
- other — если термин не удаётся классифицировать в эти категории

Требования:
1. Используй только один тип для каждого термина.
2. Основывайся только на определении и общих научных знаниях.
3. Не придумывай нового содержания — классифицируй строго по предоставленным данным.
4. Формат вывода — JSON-список:

```json
{{
    "terms": [
        {{
            "term": "<название>",
            "type": "<тип>"
        }},
        {{
            "term": "<название>",
            "type": "<тип>"
        }},
        ...
    ]
}}
```

Теперь классифицируй следующие сущности:

{terms_with_definitions}
